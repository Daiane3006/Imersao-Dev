# Imersão Dev 9º Edição

Nesta imersão iremos megulhar em lógica de programação enquanto criamos cinco jogos diferentes.

### Conversor de Moedas

Variáveis, operações e entrada/saída de dados criando em conversor de won para real e dólar.

### Pedra, Papel e Tesoura

Condicionais, números aleatórios e funções para desenvolver seu próprio jogo interativo.

### Salto na Ponte de Vidro

Loops, condicionais e funções para programar um jogo onde cada escolha pode ser decisiva.

### Cabo de Gerra

Arrays, loops e funções para criar um jogo dinâmico e desafiar a força do seu time.

### Quiz Interativo

Construir um quiz interativo no estilo show do milhão, usando JSON e inteligência artificial.
